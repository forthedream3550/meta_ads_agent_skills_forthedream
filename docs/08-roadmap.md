# Roadmap

## 0.1: Public foundation

- publish six skills;
- publish schemas and policy model;
- run local dry-run MCP server;
- support deterministic frequency analysis;
- support change-plan review;
- validate all skills and tests.

## 0.2: Supervised Meta operations

- complete Graph API read adapter;
- support object status changes;
- support bounded ad set budget changes;
- fetch before and after state;
- add approval signing and replay protection;
- add account allowlists;
- add client setup guides.

## 0.3: Lead quality feedback

- define CRM adapter interface;
- ingest aggregated lead outcomes;
- report qualified lead cost and stage conversion;
- detect follow-up bottlenecks;
- add privacy and retention controls.

## 0.4: Recognition and creative rotation

- add creative taxonomy;
- add rotation scheduling;
- compare fatigue by hook, format and proof type;
- add recognition-layer planning;
- add configurable alert windows.

## 1.0: Production operating layer

- remote OAuth-protected MCP deployment;
- approval UI;
- multi-account tenancy;
- durable audit storage;
- rollback automation;
- observability and incident playbooks;
- stable provider contract.
