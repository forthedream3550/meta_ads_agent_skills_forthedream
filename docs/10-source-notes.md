# Source notes

Review these primary sources before changing provider behavior or production setup:

- Meta Ads AI Connectors and Ads MCP Server developer documentation.
- Meta Advantage+ leads campaign guidance.
- Meta Performance 5 guidance.
- Meta Conversions API documentation.
- Meta Marketing API reference and version changelog.
- Model Context Protocol specification, tools and authorization sections.

Public documentation changes over time. Provider code must keep the Graph API version configurable and should add integration tests before a version upgrade.
